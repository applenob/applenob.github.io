Setuptools是基于distutils增强的工具，旨在让用户更简单地构建（build）和分发（distribute）python的package。

主要功能：

  • 在构建的时候，自动使用EasyInstall tool来下载、安装、更新其他依赖。
  • 创建Python Eggs，一种可移动的分发格式。
  • 无需MANIFEST.in文件
